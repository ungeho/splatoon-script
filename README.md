# splatoon-script

FFXIV の Splatoon 用スクリプト置き場です。

## スクリプト一覧

### 絶エヌオー討滅戦

#### EX7 Chaotic Torrent Orbs.cs

「混沌の激流」の玉取り位置を案内するスクリプトです。  
自分のロールに対応する玉を判定し、1個目と2個目で表示するガイド位置を切り替えます。

#### EX7 Void Pursuit.cs

「無の追跡」で、2回目（反時計回り）の誘導に自分が選出されたとき、自分が追従すべきプレイヤーに紐づく玉の誘導位置を案内するスクリプトです。  
対象の玉の位置を基準に、90度反時計回りの位置へ誘導します。

### 絶オメガ検証戦

#### TOP_Cross_Oversampled_Wave_Cannon.cs

「検知式波動砲」の十字式攻略における立ち位置を案内するスクリプトです。

## 前提

各ファイルは Dalamud / ECommons / Splatoon のスクリプト環境で動作することを想定しています。スクリプトを使用する際は、対象コンテンツ内で Splatoon のスクリプトとして読み込んでください。
